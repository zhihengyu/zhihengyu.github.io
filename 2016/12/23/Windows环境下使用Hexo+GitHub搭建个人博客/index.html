为什么使用Hexo搭建博客？

博主经过一段时间的研究后，越发觉得通过Hexo搭建博客真是省时省力，简单方便。尤其是Hexo可以使用Markdown解析文章，并自动生成对应的HTML静态页面，省去了编写HTML代码的麻烦，使我们可以把大部分精力放在编写每篇文章的内容上。这里就是我基于Hexo搭建的博客，废话不多说，你也来试试吧。

准备工作

在安装Hexo之前，必须先检查电脑中是否已安装下列应用程序：

  • Git
  • Node.js

如果没有安装，点击超链接进入下载界面，选择对应的系统版本下载并安装程序。
