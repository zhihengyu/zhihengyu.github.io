发表于 2017-01-14 | 分类于 技术 ， Linux

权限

d或- rwxr-xr-x
d代表目录，-代表文件，r表示read,w表示write,x表示execute

rwx r-x r-x
111 101 101
7 5 5
用户 用户组 其他组

最高权限为777

1.第一种授权方法
//给组加或减写的权限
chmod g+w 文件名或目录名
//给用户加或减写的权限
chmod u+w 文件名或目录名
//给其他组加或减写的权限
chmod o+w 文件名或目录名
2.第二种授权方法
chmod 777 文件名或目录名
//递归授权（授权目录及目录下的所有文件）
chmod 777 -R 文件名或目录名

//删除用户组
groupdel tomcat
//添加用户组
groupadd -g 701 tomcat
//添加用户
useradd -g 701 tomcat

安装软件方式

1.rpm包 相当于window里的exe安装方式，属于二进制安装方式
2.tar包 直接解压安装
3.yum安装（centOS的在线安装工具），在线安装的rpm

//改变文件所有者
chown 用户名 文件名

rpm安装常用命令:
//安装（install）
rpm -i 文件路径
//安装并显示进度
rpm -ivh 文件路径

tomcat用户下配置环境变量（用户级别）

1.cd （cd后面什么都不加，进到到tomcat用户目录下）
2.ls -la 显示所有文件（包括隐藏文件）
3.vi .bash_profile 编辑文件（JAVA_HOME/PATH）
例如：JAVA_HOME=/user/java/jdk1.7.0_67
export JAVA_HOME
PATH=$JAVA_HOME/bin:$PATH:$HOME/bin

root用户下配置环境变量(配置系统级别环境变量)
1.vi /etc/profile
修改内容：JAVA_HOME=/user/java/jdk1.7.0_67
export JAVA_HOME
PATH=$JAVA_HOME/bin:$PATH
export PATH
2.source /etc/profile 让该配置文件立即生效

MySQL安装

1.查询系统中是否有MySQL
rpm -qa | grep mysql
2.删除已有的mysql版本
rpm -e 后加查询出的rpm包名
rpm -e 后加查询出的rpm包名 –nodeps (如果提示有依赖关系看，强制删除)
3.解压tar
tar -xvf tar包 （解压到当前目录）
tar -xvf tar包 -C 指定目录 （指定解压目录解压）
4.安装rpm包（只安装server和client）
注意先安装server再安装client
5.查看mysql服务是否开启
ps -ef|grep mysql
查看所有服务
chkconfig –list
开启mysql服务
service mysql start
6.进入mysql
mysql -u root -p
查看一下数据库
show databases;
查看字符集
show variables like ‘chara%’;
7.退出MySQL
exit
修改root用户密码
mysqladmin -u root -p password
设置服务字符集为utf8
配置文件位置：/usr/share/mysql/my-small.cnf
把my-small.cnf拷贝到/etc/my.cnf
修改my.cnf文件，在[mysqld]这个下面加上：character_set_server = utf8
修改完后，重启MySQL服务
service mysql restart
8.开放端口
第一种方法：关闭防火墙（service iptables stop） 不可取
第二种方法：开端口 iptables -I INPUT -p tcp –dport 3306 -j ACCEPT
保存iptables配置 service iptables save
9.查看MySQL日志: /var/lib/mysql
10.数据库图形化软件访问需要授权，linux进入mysql
grant all privileges on . to root@’%’ identified by ‘数据库密码’ with grant option;
写入授权表
flush privileges;

Tomcat安装

1.解压tar包
tar -xvf apache-tomcat-7.0.57.tar.gz -C /opt/software/tomcat/
解压zip包
unzip
2.开端口 iptables -I INPUT -p tcp –dport 8080 -j ACCEPT
保存iptables配置 service iptables save
3.启动tomcat
cd到tomcat目录下 cd apache-tomcat-7.0.57/
cd bin
启动startup.sh文件 ./startup.sh
4.浏览器访问tomcat http://192.168.177.130(ip地址):8080

部署项目

1.从MyEclipse中导出项目war文件
2.将war文件导入linux的tomcat文件夹下的webapps文件夹中(当tomcat启动时会自动解压war文件)
3.停止再启用tomcat
