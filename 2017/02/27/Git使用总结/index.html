  8. 生成ssh密匙
    Windows:
    右键鼠标，点击Git Bash
    输入 ssh-keygen -t rsa -C ‘$your_email’,一直按enter
    注意id-rsa.pud的保存路径，在文件夹下打开，把里面的内容拷贝到GitHib sshkey设置里
    Mac:
    打开终端，输入ssh-keygen -t rsa -C ‘$your_email’
    输入 cat~/.ssh/id_rsa.pub
    把公钥的内容拷贝到gitHib sshkey
